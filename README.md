# vue-libary-common

> A template for create vue conmponent libary.

[中文文档](README.CN.md)

## Dev Setup

``` bash

# install dependencies
yarn

# build for production with minification
yarn run build

# run eslint and fix code style
npm run lint

# run all tests
yarn run test

```
